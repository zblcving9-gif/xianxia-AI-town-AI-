## 游戏概述

《修仙AI小镇》是一款融合了物理引擎、AI对话和修仙元素的模拟经营游戏。玩家将体验从练气期到元婴期的完整修仙之旅。

### 游戏特色

- **物理引擎驱动**：基于Matter.js的真实物理交互
- **AI智能对话**：集成阿里云Qwen大模型，NPC智能回应
- **11大系统联动**：建造、采集、战斗、门派、天气、对话、修炼、合成、社交、生存、灵力
- **自由探索世界**：3000×2000的游戏地图，多处灵力宝地

### 游戏目标

- 提升修为，突破境界
- 加入门派，学习功法
- 建设基地，积累资源
- 探索世界，结交NPC

## 快速上手

### 第一步：熟悉操作

```
移动：WASD 或 方向键
交互：E键（采集、对话）
攻击：空格键
建造：B键
合成：C键
修炼：X键
门派：F键
进食：G键
烹饪：R键（需火源附近）
```

### 第二步：采集初始资源

1. 找到附近的树木或灵草
2. 靠近后按 **E键** 采集
3. 获得木材、灵草等基础材料

### 第三步：建造营火

1. 按 **B键** 打开建造菜单
2. 选择"营火"（需要木材x5）
3. 点击地面放置

### 第四步：配置AI对话（可选）

1. 与NPC对话 → 点击"API设置"
2. 输入阿里云Qwen API Key
3. 保存后可进行智能对话

## 核心系统详解

### 一、修仙境界系统

游戏共有10个境界等级：

| 境界 | 修为需求 | 全属性加成 |
|------|----------|------------|
| 练气一层 | 0 | 1.0x |
| 练气二层 | 100 | 1.2x |
| 练气三层 | 300 | 1.4x |
| 筑基初期 | 600 | 2.0x |
| 筑基中期 | 1,200 | 2.5x |
| 筑基后期 | 2,000 | 3.0x |
| 金丹初期 | 4,000 | 4.0x |
| 金丹中期 | 8,000 | 5.0x |
| 金丹后期 | 15,000 | 6.0x |
| 元婴初期 | 30,000 | 8.0x |

**突破奖励**：
- 生命上限 +20
- 法力上限 +15
- 体力上限 +10
- 攻击力 +3
- 防御力 +2

### 二、灵力系统

世界中的灵力浓度影响修炼效率：

| 灵力等级 | 修炼加成 | 分布区域 |
|----------|----------|----------|
| 贫瘠 | 0.5x | 普通地面 |
| 稀薄 | 1.0x | 大部分区域 |
| 普通 | 1.2x | 一般灵脉 |
| 充裕 | 1.5x | 灵泉眼附近 |
| 浓郁 | 2.0x | 仙山福地 |
| 洞天福地 | 2.5x | 仙灵洞天 |
| 仙家圣地 | 5.0x | 特殊位置 |

**灵力提升技巧**：
1. 建造聚灵阵（灵石x10）
2. 移动到地图上的紫色光晕区域
3. 在练功场附近修炼

### 三、门派系统

四大门派各有特色：

| 门派 | 颜色 | 主要加成 | 适合玩家 |
|------|------|----------|----------|
| 青云门 | 蓝色 | 修炼+20%, 战斗+10% | 综合型 |
| 药王谷 | 绿色 | 治疗+30%, 炼丹+20% | 炼丹师 |
| 天音宗 | 紫色 | 社交+30%, 法力+20% | 交际型 |
| 万剑宗 | 金色 | 战斗+40% | 战斗型 |

**门派等级晋升**：

```
外门弟子 → 内门弟子(50贡献) → 亲传弟子(150贡献) 
→ 长老(400贡献) → 副掌门(1000贡献) → 掌门(2500贡献)
```

**获取贡献方法**：
- 采集任务：提交灵草x3 → +10贡献
- 战斗任务：消耗体力 → +20贡献
- 修炼任务：消耗法力 → +15贡献

### 四、天气系统

天气动态变化，影响游戏玩法：

| 天气 | 修炼 | 体力 | 免疫 | 特殊效果 |

| ☁️ 多云 | 1.1x | -5% | 正常 | 无 |
| 🌧️ 小雨 | 1.2x | -15% | 正常 | 视野略降 |
| 🌧️ 大雨 | 1.3x | -30% | -1/s | 视野降低 |
| 🌨️ 小雪 | 1.5x | -20% | -0.5/s | 视野略降 |
| 🌨️ 大雪 | 2.0x | -40% | -1.5/s | 视野降低 |
| 🌫️ 浓雾 | 0.8x | -10% | 正常 | 视野大降 |

### AI对话使用方法

**配置步骤**：

1. 获取API Key
   - 访问：https://dashscope.console.aliyun.com
   - 开通灵积模型服务
   - 创建API Key

2. 游戏内配置
   - 与NPC对话 → 点击"API设置"
   - 粘贴API Key → 保存
   - API Key会保存在浏览器本地

3. 开始对话
   - 点击"自由对话"
   - 输入你想说的话
   - NPC会用修仙风格回应

**对话技巧**：
- 提问关于修炼的问题获得指点
- 询问天气、灵力获得信息
- 建立好感度解锁更多功能

**提升好感**：

## 战斗与探索

### 战斗机制

**攻击计算公式**：
```
最终伤害 = (基础攻击 × 境界加成 × 功法加成 × 连击加成) - 敌方防御
连击加成 = 1 + (连击数 × 10%)，最高+100%
```

### 连击系统

| 连击数 | 伤害加成 | 建议 |
|--------|----------|------|
| 1-3 | +10%~30% | 保持节奏 |
| 4-6 | +40%~60% | 最佳输出 |
| 7-9 | +70%~90% | 高伤输出 |
| 10+ | +100% | 极限伤害 |

**连击技巧**：
- 2秒内再次攻击保持连击
- 注意体力消耗
- 配合功法使用

### 功法战斗应用

| 功法 | 战斗效果 | 使用时机 |
|------|----------|----------|
| 雷击术 | 伤害+50% | 开战爆发 |
| 云步 | 移动速度翻倍 | 追击/逃跑 |
| 回春术 | 恢复生命 | 战斗回复 |

### 探索要点

**必访地点**：
1. 仙灵洞天（最高灵力区域）
2. 各NPC位置
3. 灵石刷新点

**安全建议**：
- 保持体力在30%以上
- 随身携带疗伤丹
- 避开敌对NPC区域

## 常见问题解答

### Q1: 为什么采集提示物品栏已满？

**A**: v3版本已修复。如果仍有问题，请检查：
- 物品栏是否真的有20个不同物品
- 尝试使用或丢弃一些物品

### Q2: 如何获得更多灵石？

**A**: 
- 在灵力区域（紫色光晕）附近采集
- 完成NPC任务
- 炼制高级丹药出售

### Q3: NPC不和我说话怎么办？

**A**: 
- 确保已配置Qwen API Key
- 尝试使用"问候"选项
- 如果关系太差，送礼提升好感

### Q4: 如何快速突破境界？

**A**: 
- 在聚灵阵+练功场组合位置修炼
- 使用聚气丹
- 等待雷暴天气

### Q5: 体力不够用怎么办？

**A**: 
- 静止休息恢复体力
- 进食提升恢复速度
- 避免在恶劣天气活动

### Q6: 免疫力下降怎么办？

**A**: 
- 在营火附近取暖
- 使用固元丹
- 避免在寒冷天气外出

### Q7: 如何存档？

**A**: 当前版本暂不支持存档，请保持浏览器开启。后续版本会添加。

### Q8: 建筑可以拆除吗？

**A**: 目前不支持拆除，请谨慎选址。
