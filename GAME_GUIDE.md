# 修仙AI小镇 - 游戏指南与攻略

## 🎮 游戏简介

《修仙AI小镇》是一款基于 Matter.js 物理引擎开发的单机修仙模拟游戏。玩家将扮演一位初入修仙界的弟子，在充满灵力的小镇中修炼、探险、交友，逐步提升境界，最终成就仙途。

---

## 🕹️ 操作说明

### 移动控制
| 按键 | 功能 |
|------|------|
| **W / ↑** | 向上移动 |
| **S / ↓** | 向下移动 |
| **A / ←** | 向左移动 |
| **D / →** | 向右移动 |

### 交互操作
| 按键 | 功能 | 说明 |
|------|------|------|
| **E** | 交互 | 与NPC对话、采集资源 |
| **空格** | 攻击 | 对敌人发动攻击（支持连击） |
| **B** | 建造 | 打开建造菜单 |
| **C** | 合成 | 打开合成面板 |
| **X** | 修炼 | 打开修炼面板 |
| **F** | 门派 | 打开门派面板 |
| **G** | 进食 | 食用物品栏中的食物 |
| **R** | 烹饪 | 在火源附近烹饪生肉 |
| **ESC** | 关闭 | 关闭所有面板 |

### 鼠标操作
- **左键点击**：在建造模式下放置建筑
- **鼠标移动**：控制建造预览位置

## 📊 角色属性

### 基础属性
| 属性 | 说明 | 恢复方式 |
|------|------|----------|
| **生命** | 角色血量，归零则死亡 | 休息、丹药、食物 |
| **灵力** | 施放技能消耗 | 自动恢复（受灵力浓度影响） |
| **体力** | 移动和战斗消耗 | 静止休息恢复 |
| **免疫** | 抵抗疾病的能力 | 火源取暖、丹药 |

### 修仙属性
| 属性 | 说明 | 提升方式 |
|------|------|----------|
| **修为** | 修炼点数累计 | 修炼、任务、丹药 |
| **境界** | 修仙等级 | 修为达标自动突破 |
| **灵力浓度** | 当前位置灵力 | 移动到高灵力区域 |

## ⚔️ 战斗系统

### 战斗机制
- **攻击范围**：角色周围80像素
- **连击系统**：连续攻击累积连击数，最高+100%伤害
- **体力消耗**：每次攻击消耗5点体力
- **境界加成**：境界越高攻击力越强

### 敌对判定
NPC在以下情况会变为敌对：
- 关系值 < -50
- 门派敌对关系

### 战斗技巧
1. 注意体力管理，体力不足无法攻击
2. 利用连击系统打出高伤害
3. 学习功法增强战斗力

## 💬 NPC对话系统

### 与NPC交互
1. 靠近NPC后按 **E** 键
2. 选择对话选项：
   - **问候**：增进好感
   - **询问任务**：接取任务
   - **交易/打造/炼丹**：功能交互
   - **自由对话**：AI智能对话

### 配置AI对话
要与NPC进行真正的AI对话，需要配置阿里云Qwen API：

1. 获取API Key：
   - 访问 [dashscope.console.aliyun.com](https://dashscope.console.aliyun.com)
   - 开通服务并创建API Key

2. 配置步骤：
   - 与NPC对话 → 点击"API设置"
   - 粘贴API Key → 保存

3. 现在"自由对话"将调用qwen-turbo模型

### NPC类型
| 类型 | 功能 | 代表人物 |
|------|------|----------|
| 长老 | 任务、指点 | 李长老 |

## 🍖 生存系统

### 饥饿机制
- 饥饿值持续下降
- 饥饿时体力不恢复
- 严重饥饿会掉血

### 食物处理
| 食物 | 处理 | 效果 |
|------|------|------|
| 生肉 | 直接吃 | 60%概率食物中毒 |
| 熟肉 | 火源烹饪 | 恢复生命+饥饿 |

### 疾病系统
| 疾病 | 原因 | 效果 |
|------|------|------|
| 食物中毒 | 吃生肉 | 持续掉血、免疫下降 |
| 感冒 | 天气寒冷 | 移动减速 |

**治疗方法**：
- 高免疫力可自愈
- 使用固元丹
- 火源附近恢复

## 💡 新手攻略

### 第一步：资源采集
1. 移动到树木或灵草附近
2. 按 **E** 键采集
3. 收集足够的木材和灵草

### 第二步：建造营地
1. 按 **B** 键打开建造
2. 先建一个营火（木材x5）
3. 营火旁边可以烹饪

### 第三步：加入门派
1. 按 **F** 键查看门派
2. 选择适合的门派加入
3. 完成门派任务获取贡献

### 第四步：修炼提升
1. 移动到灵力浓度高的区域
2. 按 **X** 键开始修炼
3. 持续修炼直到突破境界

### 第五步：探索社交
1. 配置Qwen API
2. 与NPC自由对话
3. 探索游戏剧情

## ❓ 常见问题

**Q: 如何保存游戏进度？**
A: 当前版本暂不支持存档，请保持浏览器页面开启。

**Q: NPC为什么不理我？**
A: 需要先配置Qwen API Key才能使用AI对话功能。

**Q: 如何恢复免疫力？**
A: 在火源附近站立即可恢复，或使用固元丹。

**Q: 境界突破后有什么变化？**
A: 突破后获得生命、法力、攻击、防御等属性提升。

**Q: 资源会刷新吗？**
A: 是的，资源采集完毕后会在一定时间后重生。
